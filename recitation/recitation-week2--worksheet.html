Integrating factors: An alternative to “variation of parameters” for first-order linear ODEs.

This alternative is outlined briefly on (Nitecki and Guterman 1992, sec. 1.3, p. 28).

Consider the first order linear ODE \[(\clubsuit) \quad \dfrac{dx}{dt} + r(t)x = q(t)\]

Put \(\rho(t) = e^{R(t)}\) where \(\displaystyle R(t) = \int r(t)dt\). If \(x = x(t)\) is a solution to \((\clubsuit)\), consider the function \(u(t) = \rho(t)x(t)\).

The product rule shows that \(\dfrac{du}{dt} = \dfrac{d}{dt}\left[\rho(t)x(t)\right] = \rho(t)x' + r(t)\rho(t)x\). Multiplying \((\clubsuit)\) by \(\rho(t)\) yields the equality \[\rho(t)x' + r(t)\rho(t)x = \rho(t)q(t),\] and we conclude that \(u(t) = \rho(t)x(t)\) is a solution to the new ODE \[(\diamondsuit) \quad \dfrac{du}{dt} = \rho(t)q(t).\]

We can now find all solutions \(u(t)\) to \((\diamondsuit)\) simply by integration: \[u(t) = \int \rho(t) q(t)dt\] and then we may recover the solutions \(x(t)\) to \((\clubsuit)\) simply by multiplying the solutions \(u(t)\) by \(\rho(t)^{-1} = 1/\rho(t)\).

The function \(\rho(t)\) is called an integrating factor.
